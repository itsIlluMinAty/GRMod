## Features

### SDK
- Contains the Source SDK without needing to pay for CS2 Prime!

### csgo_gc
- Added `csgo_gc` to fix the in-game inventory system.
- The default `inventory.txt` file now comes with the **Global Offensive Badge** and **CS:GO Music Kit** rather than **Karambit | Fade** by default.
- Added flo's `csgo_gc` inventory editor with the base install for easily adding skins without an external download.

### UI Changes
- Removed the CS:GO 360 Stats button and other menu elements that were deprecated when CS:GO initially went offline.
- Removed the Danger Zone button from the "Practice with Bots" tab due to the game mode not being functional offline (you can still play it on Dedicated Servers).
- Removed broken video options.
- Modified select UI strings.
- Server Browser button is now in the main menu sidebar for convenience.
- Added a Viewmodel tab to the game settings for convenience.
- Added a "Scramble Teams" button to the call vote screen.
- Added a "Clear Inventory Images" button to the inventory screen.
- Removed all Community Market buttons.
- The TV button now opens the Demo Viewer instead of the "Watch Matches and Tournaments" page.
- Scrollable map list in the change map vote screen.
- Uniform map icon sizes in the change map vote screen.
- Restored the rank profile card (rank is editable through `csgo_gc\config.txt`).

### Map Pool Changes
- Added **Insertion II**, **Pre-Halloween Cobblestone**, and **Seaside** to the offline Casual and Competitive map pools.
- Added **Safehouse** and **Rialto** to the offline Wingman map pool.
- Added **Aztec** and **Dust** to the offline Casual and Deathmatch map pools.

## Installation

### Recommended: Automatic Installer (Windows Only)
1. Download [CS:GO from Steam](https://store.steampowered.com/app/4465480/CounterStrikeGlobal_Offensive/).
2. Download the latest release of the installer [here](https://github.com/Global-Retake/GRModInstaller/releases).
3. Open the Installer and follow the steps.
4. Click **Install**.
5. Launch the game from Steam.

### Manual Install (Windows & Linux)
1. Download [CS:GO from Steam](https://store.steampowered.com/app/4465480/CounterStrikeGlobal_Offensive/).
2. Download the latest release for your platform from the [Releases Page](https://github.com/itsIlluMinAty/GRMod/releases).
3. Navigate to the game's installation directory.
4. Extract the contents of the downloaded archive to your game directory, replacing files when prompted.
5. Launch the game from Steam.
